## 문제

무한히 큰 배열에 다음과 같이 분수들이 적혀있다.

|   |   |   |   |   |   |
|---|---|---|---|---|---|
|1/1|1/2|1/3|1/4|1/5|…|
|2/1|2/2|2/3|2/4|…|…|
|3/1|3/2|3/3|…|…|…|
|4/1|4/2|…|…|…|…|
|5/1|…|…|…|…|…|
|…|…|…|…|…|…|

이와 같이 나열된 분수들을 1/1 → 1/2 → 2/1 → 3/1 → 2/2 → … 과 같은 지그재그 순서로 차례대로 1번, 2번, 3번, 4번, 5번, … 분수라고 하자.

X가 주어졌을 때, X번째 분수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 X(1 ≤ X ≤ 10,000,000)가 주어진다.

## 출력

첫째 줄에 분수를 출력한다.

- 처음 떠올린 접근: 한쪽이 증가하면 한쪽은 감소하고 감소하는 쪽은 0이 되면 거기에 1을 더하고 방향을 바꿔서 계속 진행

## 🚀 구현

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int X = sc.nextInt();

        int a = 1, b = 1;

        // 방향: true = ↗, false = ↙
        boolean dir = true;

        for (int i = 1; i < X; i++) {

            if (dir) { // ↗ (a--, b++)
                a--;
                b++;

                if (a == 0) {
                    a = 1;
                    dir = false;
                }

            } else { // ↙ (a++, b--)
                a++;
                b--;

                if (b == 0) {
                    b = 1;
                    dir = true;
                }
            }
        }

        System.out.println(a + "/" + b);
    }
}
```

## ✅ 개선 포인트 (리팩토링)

- 더 나은 방법: 누적합이라는 방법으로 n번째 대각선에 n개의 요소가 있다는 것을 이용하여 x번째 이전의 대각선 갯수를 알아내 누적합으로 n(n+1)/2 로 갯수를 알아내고 x에서 이 값을 빼어서 현재 대각선에 몇번째를 가르키는지 알아냄. 또 n이 홀수냐 짝수냐에 따라 방향이 바뀌는 것으로 이것도 적용.
- 다른 사람 풀이 참고:
- 개선된 코드:
```java
import java.util.*;

public class Main {
	public static void main(String[] args) throws Exception {
		Scanner sc = new Scanner(System.in);

		int x = sc.nextInt();
		
		int n = 0;
		
		while(n*(n+1)/2<x) {
			n++;
		}
		
		int prev = n*(n-1)/2;
		int k = x - prev;
		
		int a,b;
		
		if(n%2==0) {
			a = k;
			b = n-k+1;
		} else {
			a = n-k+1;
			b = k;
		}
		
		System.out.print(a + "/" + b);
		
		
		sc.close();
	}
}

```
