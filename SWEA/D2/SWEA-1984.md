## 💡 접근 방법 (아이디어)

- 처음 떠올린 접근: 
	입력을 받으면서 최솟값과 최댓값을 동시에 갱신하고, 전체 합을 구한 뒤 최솟값과 최댓값을 제외하여 평균을 계산

- 핵심 아이디어: 
	전체 합에서 `(최솟값 + 최댓값)`을 빼면 나머지 8개의 합이 됨 → 이를 8로 나누어 평균 계산

- 사용한 알고리즘: 
	단순 반복문 + 조건 비교 (완전 탐색 수준)

- 시간 복잡도 예상:
	  O(T × 10) → 사실상 O(T)

- 공간 복잡도 예상:
	  O(1) (배열 사용 시 O(10))

## 🚀 구현

```java
import java.util.*;

public class Main {
	public static void main(String[] args) throws Exception {
		Scanner sc = new Scanner(System.in);

		int t = sc.nextInt();
		
		for(int T = 1 ; T<=t ; T++) {
			int[] n = new int[10];
			int min=10000, max=0;
			int sum = 0;
			int result=0;
			double avg=0;
			
			for(int i=0 ; i<10 ; i++) {
				
				n[i] = sc.nextInt();
				
				if(n[i]<min) {
					min = n[i];
				}
				if(n[i]>max) {
					max = n[i];
				}
				
			}
			
			for(int j=0 ; j<10 ; j++) {
				sum += n[j];
			}
			
			sum = sum-(min+max);
			result = sum/8;
			avg = sum/8.;
			
			if(avg-result>=0.5) {
				result+=1;
			}

			System.out.println("#" + T + " " + result);
			
		}
		sc.close();
	}
}
```

## 🔍 풀이 설명

- 코드 흐름 설명: 
	1. 테스트 케이스 수 입력
	2. 각 테스트 케이스마다 10개의 숫자 입력
	3. 입력과 동시에 최솟값, 최댓값 갱신
	4. 전체 합을 구한 뒤 최솟값과 최댓값 제거
	5. 남은 값(8개)의 평균 계산
	6. 소수점 반올림 처리 후 출력

- 중요한 포인트:
	- `sum - (min + max)`로 불필요한 값 제거
	- 정수 나눗셈과 실수 나눗셈을 분리해서 반올림 처리

- 실수했던 부분:
	- 처음부터 `sum`을 입력 과정에서 누적하지 않고 따로 반복문을 한 번 더 사용함

## ✅ 개선 포인트 (리팩토링)

- 더 나은 방법:
	- 배열 없이 입력받으면서 `sum`, `min`, `max`를 동시에 처리 가능
	- `Math.round()`를 사용하면 반올림 로직 단순화 가능
  
- 다른 사람 풀이 참고:
  
- 개선된 코드:
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int t = sc.nextInt();

        for (int T = 1; T <= t; T++) {

            int min = 10000;
            int max = 0;
            int sum = 0;

            for (int i = 0; i < 10; i++) {
                int num = sc.nextInt();
                sum += num;

                if (num < min) min = num;
                if (num > max) max = num;
            }

            sum -= (min + max);

            int result = (int) Math.round(sum / 8.0);

            System.out.println("#" + T + " " + result);
        }
    }
}
```

## 📚 배운 점

- 새로 알게 된 개념:
	  `Math.round()`를 활용한 반올림 처리
	  
- 기억해야 할 패턴:
	- 입력과 동시에 처리 (min, max, sum 같이 처리)
	- 불필요한 반복 제거
  
- 다음에 적용할 점:
	- 배열 없이 처리 가능한 경우는 최대한 단순화
	- 수동 구현보다 표준 라이브러리 우선 사용
