## 💡 접근 방법 (아이디어)

- 처음 떠올린 접근:
	- 입력을 보니 각 정점 번호와 연결된 자식 노드 정보가 존재하여 트리 구조라고 판단했습니다.
	- 연산자는 자식 노드를 가지고 있고 숫자는 리프 노드라는 점을 이용했습니다.
  
- 핵심 아이디어:
	- 각 노드의 값을 저장하고 왼쪽/오른쪽 자식 정보를 배열로 저장한 뒤 DFS 재귀를 이용하여 아래 노드부터 계산합니다.
  
- 사용한 알고리즘:
	- DFS
  
- 시간 복잡도 예상:
	- O(N)
  
- 공간 복잡도 예상:
	- O(N)

## 🚀 구현

```java
public class Main {
	static String[] value;
	static int[] left;
	static int[] right;
	
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        for(int t=1 ; t<=10 ; t++) {
        	int n = sc.nextInt();
        	value = new String[n+1];
        	left = new int[n+1];
        	right = new int[n+1];
        	
        	for(int i=0 ; i<n ; i++) {
        		int from = sc.nextInt();
        		value[from] = sc.next();
        		
        		if(isOperator(value[from])) {
        			left[from] = sc.nextInt();
        			right[from] = sc.nextInt();
        		}
        		
        	}
        	
        	int result = dfs(1);
        	
        	System.out.println("#" + t + " " + result);
        	
        }
        
        sc.close();
        
    }
    
    static int dfs(int node) {
    	
    	if(left[node] == 0 && right[node] ==0) {
    		return Integer.parseInt(value[node]);
    	}
    	
    	int leftValue = dfs(left[node]);
    	int rightValue = dfs(right[node]);
    	
    	String op = value[node];
    	
    	if(op.equals("+")) {
    		return leftValue + rightValue;
    	}
    	
    	if(op.equals("-")) {
    		return leftValue - rightValue;
    	}
    	
    	if(op.equals("*")) {
    		return leftValue * rightValue;
    	}
    	
    	return leftValue / rightValue;
    }
    
    static boolean isOperator(String op) {
        return op.equals("+") ||
               op.equals("-") ||
               op.equals("*") ||
               op.equals("/");
    	
    }
}
```

## 🔍 풀이 설명

- 코드 흐름 설명:
	1. 각 노드의 정보를 입력받습니다.
	2. 연산자 노드이면 왼쪽/오른쪽 자식 번호를 저장합니다.
	3. 숫자 노드는 리프 노드이므로 값만 저장합니다.
	4. DFS를 통해 자식 노드부터 계산합니다.
	5. 현재 노드가 연산자이면 왼쪽 결과와 오른쪽 결과를 연산합니다.

- 중요한 포인트:
	- 배열로 트리 표현
		- 현재노드 값 저장 -> `value[node]`
		- 왼쪽 / 오른쪽 자식 저장 -> `left[node]` , `right[node]`
	- 리프 노드 판별
		- 자식이 없으면 숫자 노드입니다.
		  `if(left[node] == 0 && right[node] ==0) {return Integer.parseInt(value[node]);}`
	- 후위순회 방식 계산
		- 자식 노드를 계산한 뒤 현재 노드를 계산합니다.

```java
int leftValue = dfs(left[node]);
int rightValue = dfs(right[node]);
```
  
- 실수했던 부분:
	- 문자열 비교 실수
		- `value[from] == "+"` 처럼 비교하면 안됨.
		- `value[from].equals("+")` 형태로 해야함.
