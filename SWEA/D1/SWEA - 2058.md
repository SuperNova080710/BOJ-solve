## 💡 접근 방법 (아이디어)

- 처음 떠올린 접근:
	- 입력받은 숫자를 한 자리씩 분리해서 모두 더하면 된다고 생각했습니다.
  
- 핵심 아이디어:
	- 문자열로 입력받은 뒤 각 문자를 숫자로 변환하여 누적합을 계산했습니다.
  
- 사용한 알고리즘:
	- 단순 반복문
  
- 시간 복잡도 예상:
	- O(N)  
		(N = 숫자의 자릿수)
  
- 공간 복잡도 예상:
	- O(1)

---

## 🚀 구현

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String n = sc.next();
        int sum = 0;

        for(int i=0 ; i<n.length() ; i++) {
        	sum += n.charAt(i) - '0';
        }
        
        System.out.print(sum);
        
    	sc.close();
    }
}
```

## 🔍 풀이 설명

- 코드 흐름 설명:
	1. 숫자를 문자열로 입력받습니다.
	2. 문자열의 각 문자를 반복문으로 순회합니다.
	3. `charAt(i)`로 문자 하나를 가져옵니다.
	4. `'0'`을 빼서 실제 숫자로 변환합니다.
	5. 변환한 값을 `sum`에 누적합니다.
	6. 최종 합계를 출력합니다.

- 중요한 포인트:
	- `'0'`을 빼면 문자 형태의 숫자를 정수로 변환할 수 있습니다.

## ✅ 개선 포인트 (리팩토링)

- 더 나은 방법:
	- 현재 코드도 충분히 간결하고 효율적입니다.
	- 다만 변수명을 조금 더 명확하게 바꿀 수 있습니다.
  
- 다른 사람 풀이 참고:
	- `toCharArray()`를 사용하는 방식도 자주 사용됩니다.
  
- 개선된 코드:
  
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        String number = sc.next();
        int sum = 0;

        for(char c : number.toCharArray()) {
            sum += c - '0';
        }

        System.out.println(sum);
    }
}
```
