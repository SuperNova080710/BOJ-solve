## 💡 접근 방법 (아이디어)

- 처음 떠올린 접근:
	- 문자열을 연/월/일로 잘라서 각각 정수로 변환한 뒤, 월마다 가능한 날짜 범위를 직접 검사했습니다.
  
- 핵심 아이디어:
	- - `substring()`으로 연, 월, 일을 분리합니다.
	- 월에 따라 최대 일 수를 조건문으로 검사합니다.
	- 유효하지 않으면 `-1`을 출력합니다.
  
- 사용한 알고리즘:
	- 구현(Implementation)
	- 조건 분기(Conditional Branching)
  
- 시간 복잡도 예상:
	- O(T)
  
- 공간 복잡도 예상:
	- O(T)

## 🚀 구현

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();
        
        for(int t=1 ; t<=T ; t++){
	        String s = sc.next();
	        int y,m,d;
	        String result;
	        String year,month,day;
	
	        y = Integer.parseInt(s.substring(0, 4));
	        m = Integer.parseInt(s.substring(4, 6));
	        d = Integer.parseInt(s.substring(6, s.length()));

	        if(y<1000) {
	        	year = "0" + Integer.toString(y);
	        } else {
	        	year = Integer.toString(y);
	        }
	        
	        if(m<10) {
	        	month = "0" + Integer.toString(m);
	        } else {
	        	month = Integer.toString(m);
	        }
	        
	        if(d<10) {
	        	day = "0" + Integer.toString(d);
	        } else {
	        	day = Integer.toString(d);
	        }
	        
	        result = year + "/" + month + "/" + day;
	        
	        if(m==2) {
	        	if(d>28) {
	        		result = "-1";
	        	}
	        } else if(m==1 || m==3 || m==5 || m==7 || m==8 || m==10 || m==12) {
	        	if(d>31) {
	        		result = "-1";
	        	}
	        } else if(m==4 || m==6 || m==9 || m==11) {
	        	if(d>30) {
	        		result = "-1";
	        	}
	        } else {
	        	result = "-1";
	        }
	        
	        System.out.println("#" + t + " " + result);
        }
        
    	sc.close();
    }
}
```

## 🔍 풀이 설명

- 코드 흐름 설명:
	1. 테스트 케이스 개수를 입력받습니다.
	2. 날짜 문자열을 입력받습니다.
	3. `substring()`으로 연/월/일을 분리합니다.
	4. `Integer.parseInt()`로 숫자로 변환합니다.
	5. 월별 최대 날짜를 조건문으로 검사합니다.
	6. 올바른 날짜면 `YYYY/MM/DD` 형식으로 출력합니다.
	7. 잘못된 날짜면 `-1`을 출력합니다.

- 중요한 포인트:
	- `substring(6, s.length())`는 사실 `substring(6)`으로도 가능합니다.
	- 월마다 최대 일 수가 다르기 때문에 조건 분기가 핵심입니다.
	- 출력 형식을 정확하게 맞춰야 합니다.
  
- 실수했던 부분:
	- 현재 코드에서는 `d <= 0`인 경우를 검사하지 않습니다.
	- 예를 들어 `20230100` 같은 입력은 잘못된 날짜인데 통과합니다.
	- `m <= 0`도 검사하는 것이 안전합니다.

## ✅ 개선 포인트 (리팩토링)

- 더 나은 방법:
	- 월별 최대 날짜를 배열로 관리하면 조건문이 줄어듭니다.
	- `String.format()`을 사용하면 `"0"`을 직접 붙이지 않아도 됩니다.
  
- 다른 사람 풀이 참고:
	- 대부분 `int[] days = {31,28,31,30...}` 형태로 구현합니다.
	- 코드 길이가 짧고 유지보수가 쉽습니다.
  
- 개선된 코드:
  
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int T = sc.nextInt();

        int[] days = {0,31,28,31,30,31,30,31,31,30,31,30,31};

        for(int t=1 ; t<=T ; t++) {

            String s = sc.next();

            int year = Integer.parseInt(s.substring(0,4));
            int month = Integer.parseInt(s.substring(4,6));
            int day = Integer.parseInt(s.substring(6,8));

            String result = "-1";

            if(month >= 1 && month <= 12) {
                if(day >= 1 && day <= days[month]) {
                    result = String.format("%04d/%02d/%02d",
                            year, month, day);
                }
            }

            System.out.println("#" + t + " " + result);
        }

        sc.close();
    }
}
```

## 📚 배운 점

- 새로 알게 된 개념:
	- `substring()`으로 문자열을 원하는 범위만 잘라낼 수 있습니다.
	- `String.format()`으로 자리수를 맞출 수 있습니다.
  
- 기억해야 할 패턴:
	- 문자열 분리
	- 숫자 변환
	- 범위 검사  
	    순서로 해결합니다.
  
- 다음에 적용할 점:
	- 반복되는 조건문은 배열이나 자료구조로 단순화할 수 있는지 먼저 생각하기.
	- 입력 범위의 최소값(`0 이하`)도 반드시 검사하기.
