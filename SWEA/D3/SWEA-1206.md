## 💡 접근 방법 (아이디어)

- 처음 떠올린 접근:
	- 기준 건물을 중심으로 좌우 2칸씩 확인하여, 해당 건물이 더 높으면 조망권이 있다고 판단
  
- 핵심 아이디어:
	- 기준 건물보다 좌우 2칸 내의 건물들 중 **최댓값보다 높을 때만** 조망권 확보
		- 확보된 층 수 = `현재 높이 - 주변 최대 높이`
  
- 사용한 알고리즘:
	- 단순 구현 (Brute Force)
  
- 시간 복잡도 예상:
	- O(N)
  
- 공간 복잡도 예상:
	- O(N)

## 🚀 구현

```java
import java.util.*;

public class Main {
	public static void main(String[] args) throws Exception {
		Scanner sc = new Scanner(System.in);
		
		for(int T = 1 ; T<=10 ; T++) {		// 10번 반복하는 구간
			int n = sc.nextInt();			// 건물 개수 입력
			int[] floor = new int[n];		// 건물 층수
			int vr =0;						// 조망권이 유효할 시에 계산
			
			for(int i=0 ; i<n ; i++) {
				floor[i] = sc.nextInt();
			}
			
			for(int j=2 ; j<n-2 ; j++) {	// 처음과 마지막 2개 제외한 반복문
				boolean vrCheck = true;		// 조망권이 유효한지 확인하는 변수
				int maxFloor = 0;

				for(int k=j-2 ; k<=j+2 ; k++) {			// j번째 건물을 기준으로 앞뒤 건물 2개씩 비교하는 반복문
					if(j!=k && maxFloor<floor[k]) {		// j번째 건물을 제외한 최고 층수
						maxFloor = floor[k];
					}
					
					if(floor[j]>floor[k] || j==k) {		// j번째 건물이거나 j번째 건물 층수가 k번째 건물보다 층수가 높으면 통과
						continue;
					} else if(floor[j]<=floor[k]) {		// j번째 건물이 k번째 건물보다 낮으면 조망권이 유효하지 않음
						vrCheck = false;
					}
					
				}
				
				if(vrCheck) {							// 조망권이 유효한 경우 
					vr += floor[j] - maxFloor;			// j번째 건물의 층수에서 j번째를 제외한 최고 층수를 빼서 조망권이 확보된 층수 계산
				}
				
			}
			
			System.out.println("#" + T + " " + vr);
		}
		
		sc.close();
	}
}
```

## 🔍 풀이 설명

- 코드 흐름 설명:
	1. 테스트케이스 10번 반복
	2. 건물 높이 배열 입력
	3. 양 끝 2칸 제외하고 순회
	4. 좌우 2칸의 최대 높이 계산
	5. 현재 건물이 더 높으면 차이만큼 더함

- 중요한 포인트:
	- "모든 건물보다 높아야 한다" 가 아니라 **주변 최대값보다 높으면 된다**
	- 조건 판단을 단순화하면 코드가 크게 줄어든다
  
- 실수했던 부분:
	- boolean으로 조망 가능 여부를 따로 체크
	- 불필요한 반복문과 조건문 사용

## ✅ 개선 포인트 (리팩토링)

- 더 나은 방법:
	- 주변 4칸중 최대값만 구해서 비교
  
- 다른 사람 풀이 참고:
  
- 개선된 코드:
  
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        for(int T = 1; T <= 10; T++) {
            int n = sc.nextInt();
            int[] floor = new int[n];
            int result = 0;

            for(int i = 0; i < n; i++) {
                floor[i] = sc.nextInt();
            }

            for(int i = 2; i < n - 2; i++) {
                int max = Math.max(
                        Math.max(floor[i-1], floor[i-2]),
                        Math.max(floor[i+1], floor[i+2])
                );

                if(floor[i] > max) {
                    result += floor[i] - max;
                }
            }

            System.out.println("#" + T + " " + result);
        }

        sc.close();
    }
}
```

## 📚 배운 점

- 새로 알게 된 개념:
	- **전체 비교** 대신 **최댓값 비교**로 문제 단순화 가능
  
- 기억해야 할 패턴:
	- 주변값 비교문제 -> **max/min**으로 축소
  
- 다음에 적용할 점:
	- 조건을 수식으로 단순화하는 습관
