## 💡 접근 방법 (아이디어)

- 처음 떠올린 접근:
    - 문자열을 한 글자씩 비교하면서 같은 문자의 개수를 세자고 생각함.
- 핵심 아이디어:
    - 같은 위치의 문자끼리 비교하여 일치하면 count 증가.
- 사용한 알고리즘:
    - 문자열 순회 (Brute Force)
- 시간 복잡도 예상:
    - O(N)
- 공간 복잡도 예상:
    - O(1)

---

## 🚀 구현

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();
        
        for(int t=1 ; t<=T ; t++) {
            int length = sc.nextInt();
            String a = sc.next();
            String b = sc.next();
            int count = 0;
            
            for(int i=0 ; i<length ; i++) {
                if(a.charAt(i) == b.charAt(i)) {
                    count++;
                }
            }
            
            System.out.println("#" + t + " " + count );
        }
        
        sc.close();
    }
}
```
